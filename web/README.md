# DecibelDesigns - Next.js Web Application

This is the primary web application for DecibelDesigns, built with **Next.js 14**, **Tailwind CSS**, and **Framer Motion**.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Material Symbols Outlined](https://fonts.google.com/icons)
- **Deployment**: Optimized for Vercel

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Pages

- `/`: Homepage (High-impact Hero + Expertise)
- `/services`: Services Overview (Bento Grid layout)
- `/portfolio`: Portfolio Gallery (Curated projects)
- `/contact`: Contact Architecture (Glass-morphism forms)

## Design tokens

The site uses a premium **Noir/Copper/Gold** color palette defined in `tailwind.config.ts`.
